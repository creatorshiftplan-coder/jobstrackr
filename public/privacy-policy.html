Privacy Policy for JobsTrackr

Last updated: January 24, 2026

JobsTrackr values your privacy and is committed to protecting your personal information. This Privacy Policy explains how we collect, use, store, and protect your data when you use the JobsTrackr mobile or web application.

1. Information We Collect

We collect information only to provide core app functionality.

a) Personal Information

When you create an account or update your profile, we may collect:

  • Name
  • Date of birth
  • Gender
  • Category
  • Email address
  • Phone number
  • Address details (optional)

b) Documents Uploaded by Users

Users may upload government-related documents such as certificates for profile updates. These documents are processed only for text extraction (OCR) and profile completion.

c) App Usage Information

We may collect limited usage data such as:

  • Exams tracked by the user
  • Features used within the app
  • App performance and error logs

2. How We Use Your Information

We use your information strictly to:

  • Track government exams and job notifications
  • Update user profiles using OCR (only after user confirmation)
  • Assist in form filling using saved profile data
  • Provide syllabus and expected cut-off insights
  • Send important exam-related notifications

We do not sell or rent user data.

3. OCR and AI Usage

  • OCR is used only to extract text from user-uploaded documents
  • AI features are used to provide expected cut-off estimates and syllabus assistance
  • All extracted information is shown to the user for review before being saved
  • No document is processed without user action

4. Data Storage and Security

  • User data is securely stored using encrypted cloud services (such as Supabase)
  • Industry-standard security measures are applied
  • Access to user data is restricted to authorized systems only

5. Data Sharing

We do not share personal data with third parties except:

  • When required by law
  • For essential infrastructure services necessary to run the app securely

6. User Control and Data Deletion

  • Users can update their profile information at any time
  • Users can request account and data deletion directly from the app or by contacting us
  • Upon deletion request, all personal data will be permanently removed within a reasonable time

7. Children's Privacy

JobsTrackr is intended for users aged 18 years and above. We do not knowingly collect personal data from children.

8. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated revision date.
